Enables Gaug.es on all pages.

== Description ==

This plugin adds the required javascript for Gaug.es web analytics tracking.

For more information visit:

[Gaug.es](http://get.gaug.es)

== Installation ==

1. Upload `gauges` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add the Gaug.es Site ID from your Gaug.es site (find it on the 'Tracking Code' page) to the settings (Admin > Settings > Gaug.es)
